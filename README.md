# CodeHub

CodeHub is a lightweight, personal snippet manager designed for web developers. It allows you to organize, search, and store your favorite HTML, CSS, and JavaScript snippets in one central place.



## Features

* **Categorized Storage:** Organise your snippets into HTML, CSS, JS, and TS tabs.
* **Live Search:** Find any snippet instantly using the global search bar.
* **Persistent Data:** Your snippets are saved locally in your browser (LocalStorage).
* **Interactive UI:** Smooth navigation with a sliding indicator and mobile-friendly scrolling.
* **Modern Design:** A sleek, dark-themed interface built for developers.

## Built With

* **HTML5:** Semantic structure for high accessibility.
* **CSS3:** Custom properties (variables), Flexbox/Grid, and smooth animations.
* **JavaScript (ES6+):** Dynamic DOM manipulation and LocalStorage integration.
